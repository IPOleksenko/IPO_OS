# IPO_OS

## ⚙️ Compilation Commands

### 🪩 To install all the necessary dependencies, run
```bash
./tools/install.sh
```

### 🧹 Clean the Build Directory
```bash
sudo make clean
```

### 🔧 Build the Project
```bash
sudo -E make build
```

### ▶️ Run the Project
```bash
sudo make run
```

### ⚒️ Clean, Build and Run (All-in-One)
```bash
sudo -E make
```
